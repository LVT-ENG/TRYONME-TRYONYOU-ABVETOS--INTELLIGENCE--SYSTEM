# Product: TRYONYOU (Divineo v7)

**Goal**: To provide an AI-Powered Virtual Try-On Platform where beauty lives in movement.

## Core Features

*   **Virtual Try-On (Wardrobe)**: Allow users to try on clothes virtually.
*   **3D Avatar (MyAvatar)**: Users can create a digital twin.
*   **AI Stylist (Ask Peacock)**: AI chat assistant for style advice.
*   **Showroom**: Curated looks gallery.
*   **Glow-Up**: AI style transformation.
*   **Brands**: Browse and filter fashion brands.

## UX/UI

*   **Glass Morphism**: Modern UI with blur effects.
*   **Interactive 3D**: Three.js powered avatar interaction.
*   **Responsive**: Mobile-first design.
