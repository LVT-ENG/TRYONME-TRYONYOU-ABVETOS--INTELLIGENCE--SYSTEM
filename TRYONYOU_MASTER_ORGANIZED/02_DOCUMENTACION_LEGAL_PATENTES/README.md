# TRYONYOU – ABVETOS – ULTRA–PLUS–ULTIMATUM

Sistema de Inteligencia de Moda Patentado | PCT/EP2025/067317

## Descripción

Este proyecto es una implementación de front-end para el ecosistema de Inteligencia de Moda Emocional **TRYONYOU**. Permite a los usuarios generar un gemelo digital, recibir looks seleccionados por IA y experimentar un nuevo paradigma en la moda y el comercio minorista.

La interfaz está diseñada con una estética "Vogue Tech", que combina el lujo (tonos dorados) con una sensación tecnológica y moderna (tonos antracita).

## Pila Tecnológica

*   **Framework:** React
*   **Herramientas de compilación:** Vite
*   **Estilos:** Tailwind CSS
*   **Iconos:** Lucide React

## Instalación

1.  Clona el repositorio:
    ```bash
    git clone https://github.com/LVT-ENG/TRYONME-TRYONYOU-ABVETOS--INTELLIGENCE--SYSTEM.git
    ```
2.  Navega al directorio del proyecto:
    ```bash
    cd TRYONME-TRYONYOU-ABVETOS--INTELLIGENCE--SYSTEM
    ```
3.  Instala las dependencias:
    ```bash
    npm install
    ```

## Uso

### Iniciar el servidor de desarrollo

Para ejecutar la aplicación en modo de desarrollo con recarga en caliente, usa el siguiente comando:

```bash
npm run dev
```

Esto iniciará un servidor de desarrollo local, generalmente en `http://localhost:5173`.

### Construir para producción

Para crear una compilación optimizada para producción, ejecuta:

```bash
npm run build
```

Los archivos compilados se generarán en el directorio `dist/`.

### Vista previa de la compilación de producción

Para obtener una vista previa de la compilación de producción localmente, usa:

```bash
npm run preview
```
