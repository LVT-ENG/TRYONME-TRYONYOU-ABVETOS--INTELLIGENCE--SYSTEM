## üîë Key Concepts

### The Three Outfit Variations
1. **Bridal Chiffon Gown** - White/ivory, ethereal, romantic
2. **Black Widow Couture** - Black, structured, dramatic
3. **Red Satin Evening** - Ruby red, elegant, flowing

### Pau the Peacock Mascot
- **Essential element** in all images
- Bottom-right corner placement
- 8-12% of image height
- White with golden reflections
- Looking toward the model

### Consistency Requirements
- SAME model across all variations
- SAME pose and expression
- SAME lighting setup
- SAME background and composition
- Only the outfit changes

### Quality Standard
- Vogue / Harper's Bazaar editorial style
- Hyper-realistic photographic quality
- 9:16 vertical format
- Luxury soft cinematic lighting
- Professional studio finish
