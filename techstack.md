# Tech Stack

## Frontend

*   **Framework**: React 18
*   **Build Tool**: Vite (v5/v7)
*   **Styling**: Tailwind CSS 3
*   **Animation**: Framer Motion 11, GSAP
*   **3D Graphics**: Three.js, @react-three/fiber, @react-three/drei
*   **Icons**: Lucide React
*   **Routing**: React Router 7 / Wouter (based on package.json)

## Backend

*   **Language**: Python
*   **Framework**: FastAPI
*   **Server**: Uvicorn
*   **AI/ML**: MediaPipe, NumPy, SciPy
*   **Validation**: Pydantic

## Infrastructure & Deployment

*   **Platform**: Vercel
*   **Deployment Script**: `TRYONYOU_SUPERCOMMIT_MAX.sh`
*   **CI/CD**: GitHub Actions
