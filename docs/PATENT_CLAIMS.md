# TRYONYOU PATENT CLAIMS
**Patent Reference:** PCT/EP2025/067317
**Title:** Modular Embeddable Digital System for Emotional Fashion Intelligence

## 4 Reivindicaciones Clave (Core Claims)

1. **Avatar 3D Paramétrico:** A modular system comprising a 3D avatar generator embeddable in third-party platforms, configured to generate a digital model of the user’s body based on input measurements.
2. **Motor Emocional (PAU):** An artificial intelligence module trained with textile and emotional datasets (PAU) that interfaces with the user to interpret and communicate suggestions based on emotional states.
3. **Pago Biométrico Dual (ABVET):** A biometric payment system using iris and voice recognition for secure transaction authorization and traceability.
4. **Economía Circular (AutoDonate):** An automated redistribution engine that redirects unused or rejected garments to donation or recycling services via a "Solidarity Wardrobe".
