# Guía de Usuario - TryOnMe / TryOnYou

Guía completa para usuarios del sistema TryOnMe / TryOnYou - AVBETOS Intelligence System.

## 🎯 ¿Qué es TryOnMe / TryOnYou?

TryOnMe / TryOnYou es un sistema inteligente de recomendaciones de moda que utiliza tecnología biométrica avanzada y algoritmos de inteligencia artificial para:

- 👗 **Recomendaciones personalizadas** basadas en tus gustos, medidas corporales y ocasiones
- 🔍 **Prueba virtual** de prendas usando realidad aumentada
- 🔐 **Autenticación biométrica** segura (voz + iris)
- 💳 **Compras seguras** con verificación biométrica
- 📊 **Análisis de tendencias** y preferencias de moda

## 🚀 Primeros Pasos

### 1. Registro en el Sistema

**Paso 1: Datos Personales**
- Nombre completo
- Correo electrónico
- Edad y género
- Preferencias iniciales de estilo

**Paso 2: Medidas Corporales**
- Altura y peso
- Medidas de pecho, cintura y cadera
- Talla habitual de ropa
- Tipo de cuerpo

**Paso 3: Registro Biométrico**
- **Muestra de voz**: Pronuncia la frase que te mostramos (15-20 segundos)
- **Escaneo de iris**: Mira fijamente a la cámara por 5 segundos
- Ambos datos se encriptan y almacenan de forma segura

### 2. Configuración de Preferencias

**Estilos Favoritos**:
- [ ] Casual/Informal
- [ ] Elegante/Formal
- [ ] Deportivo/Athleisure
- [ ] Bohemio/Boho
- [ ] Minimalista
- [ ] Vintage/Retro

**Ocasiones Frecuentes**:
- [ ] Trabajo/Oficina
- [ ] Eventos sociales
- [ ] Tiempo libre/Fin de semana
- [ ] Ejercicio/Deporte
- [ ] Viajes
- [ ] Citas especiales

**Presupuesto**:
- Rango de precio preferido por prenda
- Marcas favoritas (opcional)
- Frecuencia de compra

## 👗 Usando las Recomendaciones

### Obtener Recomendaciones

1. **Accede a tu dashboard** después de iniciar sesión
2. **Selecciona la ocasión** para la que necesitas ropa
3. **Especifica preferencias adicionales** (colores, estilos, presupuesto)
4. **Recibe hasta 20 recomendaciones personalizadas**

### Información de Cada Recomendación

Cada producto recomendado incluye:

- **Imagen del producto** en alta resolución
- **Información básica**: marca, precio, disponibilidad
- **Porcentaje de coincidencia** con tus preferencias
- **Razones de la recomendación**:
  - "Coincide con tu estilo casual preferido"
  - "Talla recomendada basada en tus medidas: M"
  - "Popular entre usuarios con gustos similares"
- **Predicción de ajuste** y talla sugerida

### Filtros y Personalización

**Filtros disponibles**:
- Categoría (tops, bottoms, vestidos, accesorios)
- Rango de precio
- Color predominante
- Marca
- Ocasión de uso
- Nivel de formalidad

## 🔍 Prueba Virtual (Try-On)

### Cómo Funciona

1. **Selecciona un producto** de las recomendaciones
2. **Toma una foto** o usa la cámara en tiempo real
3. **El sistema procesa la imagen** (3-5 segundos)
4. **Ve el resultado** de cómo te quedaría la prenda

### Mejores Resultados

**Para la foto**:
- ✅ Buena iluminación natural
- ✅ Fondo neutro (pared blanca/gris)
- ✅ Postura frontal, brazos a los lados
- ✅ Ropa ajustada que muestre tu silueta
- ❌ Evita sombras fuertes
- ❌ No uses ropa muy holgada
- ❌ Evita fondos muy decorados

**Análisis de Ajuste**:
- **Excelente** (verde): La prenda te quedará perfecta
- **Bueno** (amarillo): Buen ajuste con pequeños ajustes
- **Regular** (naranja): Puede quedar algo holgado/ajustado
- **Pobre** (rojo): No recomendado para tu talla/tipo de cuerpo

## 💳 Proceso de Compra

### Verificación Biométrica

Antes de cualquier compra, el sistema requiere verificación biométrica:

1. **Autenticación por voz**: Pronuncia tu frase de seguridad
2. **Verificación de iris**: Mira a la cámara por 3 segundos
3. **Confirmación**: El sistema verifica tu identidad (>90% de confianza)

### Métodos de Pago

- 💳 Tarjetas de crédito/débito
- 🏦 Transferencia bancaria
- 📱 Pagos móviles (Apple Pay, Google Pay)
- 🔐 Todos los pagos están protegidos con verificación biométrica

### Seguimiento de Pedidos

- ✅ Confirmación inmediata por email
- 📦 Número de seguimiento del envío
- 📱 Notificaciones push del estado
- 🔄 Actualizaciones en tiempo real en tu dashboard

## 🔐 Seguridad y Privacidad

### Protección de Datos Biométricos

- **Encriptación avanzada**: Tus datos biométricos se encriptan con AES-256
- **Almacenamiento seguro**: Los datos nunca se almacenan en texto plano
- **No compartimos datos**: Tu información biométrica nunca se comparte con terceros
- **Derecho al olvido**: Puedes eliminar todos tus datos en cualquier momento

### Control de Privacidad

**En tu perfil puedes controlar**:
- [ ] Permitir análisis de tendencias anónimo
- [ ] Recibir recomendaciones por email
- [ ] Compartir datos de ajuste para mejorar algoritmos
- [ ] Notificaciones de ofertas personalizadas

### Autenticación de Dos Factores

Para mayor seguridad:
1. **Factor 1**: Credenciales (email + contraseña)
2. **Factor 2**: Verificación biométrica (voz + iris)
3. **Factor 3** (opcional): SMS o app autenticadora

## ❓ Preguntas Frecuentes

### ¿Es seguro el sistema biométrico?

Sí, utilizamos tecnología de grado militar para proteger tus datos biométricos. Los datos se procesan localmente cuando es posible y se encriptan antes de cualquier transmisión.

### ¿Qué pasa si no me gusta una recomendación?

Cada recomendación tiene botones de 👍 y 👎. Tu feedback mejora inmediatamente las futuras recomendaciones. También puedes marcar "No me interesa este estilo" para ajustes más específicos.

### ¿Puedo usar el sistema sin verificación biométrica?

No para compras, ya que es nuestro método de seguridad principal. Sin embargo, puedes explorar recomendaciones y usar la prueba virtual sin verificación biométrica.

### ¿Funciona la prueba virtual con todo tipo de ropa?

Actualmente funciona mejor con:
- ✅ Camisetas y tops
- ✅ Chaquetas y blazers
- ✅ Vestidos
- ⚠️ Pantalones (en desarrollo)
- ❌ Ropa interior y accesorios pequeños

### ¿Cómo mejoro la precisión de las recomendaciones?

1. **Completa tu perfil al 100%**
2. **Proporciona feedback** en cada recomendación
3. **Actualiza tus medidas** si cambian
4. **Marca tus productos favoritos** de vez en cuando
5. **Usa la función "no me gusta"** cuando sea apropiado

### ¿Puedo cambiar mis datos biométricos?

Sí, puedes actualizar tus muestras biométricas en Configuración > Seguridad > Actualizar Biometría. Se requiere verificación con los datos actuales.

### ¿Qué hago si el sistema no reconoce mi voz?

1. Asegúrate de estar en un ambiente silencioso
2. Habla con claridad y volumen normal
3. Usa la misma entonación que durante el registro
4. Si persiste el problema, contacta soporte para re-registrar tu voz
