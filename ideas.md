<idea>
  **Design Movement**: **Futurismo de Alta Costura (High-Fashion Futurism)**
  **Core Principles**:
  1. **Elegancia Tecnológica**: Fusionar la sofisticación clásica de Galeries Lafayette con la innovación de la IA biométrica.
  2. **Fluidez Digital**: Transiciones suaves y orgánicas que imitan el movimiento de la tela y la luz.
  3. **Minimalismo de Lujo**: Espacios amplios, tipografía refinada y elementos visuales que respiran exclusividad.
  4. **Inmersión Sensorial**: Uso de efectos visuales (blur, glow) para crear una atmósfera envolvente y moderna.

  **Color Philosophy**:
  - **Base**: Negro profundo (#0a0a0a) y Blanco puro (#ffffff) para un contraste dramático y elegante.
  - **Acento**: "Lafayette Red" (#E2001A) reinterpretado como un neón sutil o degradado para indicar interacción y tecnología.
  - **Secundario**: Plata metálica y gris pizarra para elementos estructurales, evocando espejos y superficies pulidas.
  - *Intención*: Evocar la sensación de estar en un probador exclusivo del futuro, donde la tecnología es invisible pero poderosa.

  **Layout Paradigm**:
  - **Asimetría Equilibrada**: Romper la rejilla tradicional con composiciones dinámicas que guían la vista.
  - **Capas de Profundidad**: Superposición de elementos (texto sobre imagen, interfaces flotantes) para crear dimensión.
  - **Navegación Flotante**: Menús y controles que parecen levitar sobre el contenido, manteniendo la inmersión.

  **Signature Elements**:
  - **Líneas de Escaneo Estilizadas**: Referencias visuales sutiles al escaneo biométrico (líneas finas, retículas) integradas como elementos decorativos.
  - **Tipografía Display Grande**: Títulos en mayúsculas, con mucho espaciado (tracking), que actúan como elementos gráficos.
  - **Efecto "Glassmorphism" Refinado**: Paneles semitransparentes con desenfoque de fondo para interfaces de usuario.

  **Interaction Philosophy**:
  - **Micro-interacciones Magnéticas**: Los botones y elementos interactivos responden con una "atracción" magnética al cursor.
  - **Feedback Visual Inmediato**: El escáner biométrico responde en tiempo real con partículas o ondas de luz al detectar movimiento.
  - **Scroll Parallax Suave**: Movimiento diferencial entre capas para acentuar la profundidad.

  **Animation**:
  - **Entradas Teatrales**: Los elementos aparecen con una coreografía suave (fade-in + slide-up) como modelos en una pasarela.
  - **Transiciones de Estado Líquidas**: Cambios de estado (ej. de "escaneando" a "resultado") que fluyen como mercurio.
  - **Loop de Fondo Sutil**: Fondos abstractos que se mueven imperceptiblemente para mantener la vida en la pantalla.

  **Typography System**:
  - **Display**: *Didot* o *Bodoni* (Serif moderna de alto contraste) para títulos y encabezados, evocando la moda editorial.
  - **Body/UI**: *Inter* o *Helvetica Now* (Sans-serif geométrica) para legibilidad técnica y datos biométricos.
  - *Regla*: Contraste extremo entre la elegancia de la serif y la precisión de la sans-serif.
</idea>

<idea>
  **Design Movement**: **Brutalismo Digital Refinado (Refined Digital Brutalism)**
  **Core Principles**:
  1. **Honestidad Estructural**: Mostrar la "estructura" de la tecnología (bordes visibles, datos crudos) pero con un acabado pulido.
  2. **Tipografía como Imagen**: El texto es el protagonista visual, grande, audaz y a menudo superpuesto.
  3. **Colores de Alta Energía**: Uso de colores vibrantes y saturados para transmitir innovación y juventud.
  4. **Interacción Cruda**: Botones grandes, bordes definidos y respuestas táctiles claras.

  **Color Philosophy**:
  - **Base**: Blanco hueso (#f5f5f5) o Gris cemento claro (#e0e0e0).
  - **Acento**: Azul eléctrico (#0000ff) o Verde ácido (#ccff00) para elementos de acción y escaneo.
  - **Texto**: Negro sólido (#000000) para máxima legibilidad e impacto.
  - *Intención*: Romper con la estética de lujo tradicional para atraer a una audiencia más joven y tecnológicamente consciente (Gen Z).

  **Layout Paradigm**:
  - **Grillas Visibles**: Líneas de división y cajas que estructuran el contenido de manera explícita.
  - **Desplazamiento Horizontal**: Secciones que se mueven lateralmente para una experiencia de navegación tipo "app".
  - **Elementos "Sticky"**: Información clave que se queda fija mientras el usuario hace scroll.

  **Signature Elements**:
  - **Bordes Gruesos**: Elementos UI con bordes negros de 1px o 2px.
  - **Monospace Fonts**: Uso de fuentes monoespaciadas para datos técnicos y etiquetas.
  - **Imágenes "Raw"**: Fotografía directa, sin filtros excesivos, o renders 3D con texturas visibles.

  **Interaction Philosophy**:
  - **Hover Explosivo**: Cambios drásticos de color o inversión (blanco/negro) al pasar el cursor.
  - **Cursor Personalizado**: Un cursor grande o con forma específica que cambia según el contexto.
  - **Scroll Snapping**: Desplazamiento que se "ajusta" a secciones específicas para una lectura enfocada.

  **Animation**:
  - **Cortes Directos**: Transiciones rápidas y sin suavizado excesivo.
  - **Texto en Movimiento (Marquee)**: Títulos que se desplazan horizontalmente para crear dinamismo.
  - **Glitch Sutil**: Pequeños efectos de distorsión digital al interactuar con elementos tecnológicos.

  **Typography System**:
  - **Display**: *Druk Wide* o *Monument Extended* (Sans-serif ancha y pesada) para impacto máximo.
  - **Body/UI**: *Space Mono* o *Roboto Mono* (Monospace) para la estética técnica.
  - *Regla*: Todo en mayúsculas para títulos, minúsculas para datos.
</idea>

<idea>
  **Design Movement**: **Eterealismo Biométrico (Biometric Etherealism)**
  **Core Principles**:
  1. **Luz y Transparencia**: Interfaces que parecen hechas de luz y cristal.
  2. **Formas Orgánicas**: Uso de curvas suaves y formas inspiradas en la anatomía humana y la naturaleza.
  3. **Colores Iridiscentes**: Paletas que cambian sutilmente, como una burbuja de jabón o un holograma.
  4. **Tecnología Invisible**: La interfaz desaparece para dejar protagonismo al usuario y su reflejo digital.

  **Color Philosophy**:
  - **Base**: Degradados suaves de pasteles (lila, menta, melocotón) sobre fondo blanco o negro muy suave.
  - **Acento**: Degradados holográficos para botones y áreas de escaneo.
  - **Texto**: Gris oscuro o blanco con sombras suaves para legibilidad sobre fondos complejos.
  - *Intención*: Crear una experiencia mágica y acogedora, eliminando la frialdad de la tecnología biométrica.

  **Layout Paradigm**:
  - **Diseño Flotante**: Elementos que no tocan los bordes y parecen flotar en el espacio.
  - **Navegación Circular**: Menús y controles dispuestos en arcos o círculos, imitando el campo de visión o el escaneo del iris.
  - **Espacio Negativo Activo**: El vacío se usa para enfocar la atención en el sujeto (el usuario).

  **Signature Elements**:
  - **Gradientes de Malla (Mesh Gradients)**: Fondos fluidos y coloridos que reaccionan al movimiento.
  - **Bordes Difusos**: Elementos sin bordes duros, que se funden con el fondo.
  - **Iconografía Lineal Fina**: Iconos muy delgados y elegantes.

  **Interaction Philosophy**:
  - **Respuesta Suave**: Interacciones que se sienten como tocar agua o aire.
  - **Seguimiento Ocular/Cursor**: Elementos que se inclinan o mueven sutilmente siguiendo el cursor (efecto 3D).
  - **Feedback Háptico Visual**: Pequeñas vibraciones visuales al completar acciones.

  **Animation**:
  - **Morphing**: Formas que cambian suavemente de una a otra.
  - **Efecto de Respiración**: Elementos que pulsan suavemente para indicar que el sistema está "vivo".
  - **Aparición por Desenfoque**: Elementos que pasan de borroso a nítido al entrar en escena.

  **Typography System**:
  - **Display**: *Ogg* o *Canela* (Serif caligráfica) para un toque humano y artístico.
  - **Body/UI**: *Satoshi* o *General Sans* (Sans-serif humanista) para claridad y calidez.
  - *Regla*: Uso de itálicas para énfasis emocional.
</idea>
</text>
